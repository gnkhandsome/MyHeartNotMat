## 严重级别建议

- **P0**：阻塞发布（崩溃、核心链路不可用）
- **P1**：高优先（核心体验明显异常，但有绕过路径）
- **P2**：中优先（体验瑕疵，不阻塞主流程）
- **P3**：低优先（文案/样式/细节优化）

## 状态流转建议

`Open -> In Progress -> Resolved -> Verified -> Closed`

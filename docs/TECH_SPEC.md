# 树洞小程序开发技术文档

> ⚠️ 文档说明：本文件为完整方案沉淀（含扩展设想）。
> 若进入**第一版开发（V1）**，请以 `docs/V1_HEALING_PRD.md` 为主执行文档。

## 1. 全局主题配置系统（Theme Manager）

为了支持后台下发的 12 种主题风格，所有主题色值统一抽离到独立配置文件中管理。

### 1.1 配置文件结构（`theme.config.js`）

```js
/**
 * 主题色值配置文件
 * 包含 12 种预设风格，支持从后台动态更新
 */
export const themes = {
  "classic_blue": {
    "name": "经典蓝",
    "primaryColor": "#007AFF",        // 主色调：按钮、高亮文字
    "backgroundColor": "#F5F5F7",     // 页面背景色
    "cardColor": "#FFFFFF",           // 卡片/容器背景色
    "tabBarActive": "#007AFF",        // 底部栏选中色
    "tabBarInactive": "#8E8E93",      // 底部栏未选中色
    "textColorMain": "#1D1D1F",       // 一级标题/正文色
    "textColorSub": "#86868B",        // 次要辅助文字色
    "dividerColor": "#D2D2D7",        // 分割线颜色
    "accentColor": "#FF9500",         // 强调色：特殊发布提醒
    "shadowColor": "rgba(0,0,0,0.1)"  // 阴影颜色
  }
  // ... 其他 11 种主题
}
```

### 1.2 应用方案

- **全局变量**：在 `app.js` 中获取当前主题并存入 `globalData`。
- **CSS 变量（推荐）**：在页面 `onShow` 时，通过内联样式或 `page-meta` 注入 CSS Variables，例如：
  - `--theme-color: #007AFF`
  - `--text-color-main: #1D1D1F`

### 1.3 情绪共鸣配置（新增）

在主题系统基础上，新增“情绪波形”配置，用于驱动卡片呼吸律动与同频提示。

  },
  joy: {

### 1.4 主题色与白噪音联动（Ambient Tide，新增）

将 12 种主题与 12 种自然白噪音绑定，形成“视觉 + 听觉”一体化疗愈体验。

```js
export const themeAudioMap = {
  classic_blue: {
    audioSrc: '/static/audio/sea_wave.mp3',
    label: '海浪潮汐',
    baseVolume: 0.45
  },
  forest_green: {
    audioSrc: '/static/audio/forest_birds.mp3',
    label: '林间鸟鸣',
    baseVolume: 0.42
  }
  // ... 其余主题对应自然音
}
```

联动规则建议：
- 切换主题时，音频淡入淡出（500ms~1200ms）
- 首页提供极简播放/暂停控制
- 列表滚动速度越快，环境音音量允许轻微上浮（如 +0.03 ~ +0.08）

## 2. 首页（Home Tab）

首页采用沉浸式布局，强调内容流动感与交互体验。

### 2.1 页面布局

1. **自定义顶部 Tab（Custom Header Tab）**
   - 固定在顶部（Sticky）
   - 包含“推荐 / 附近 / 最新”等分类切换

2. **内容列表（Feed List）**
   - 瀑布流或卡片式布局
   - 支持下拉刷新（`onPullDownRefresh`）与上拉加载更多

3. **底部 Toolbar**
   - 展示系统通知、搜索入口或状态统计

4. **自由拖动发布按钮（Draggable FAB）**
   - 使用 `movable-area` + `movable-view` 实现
   - 拖动范围限制在内容区，避免遮挡顶部 Tab

### 2.2 情绪共鸣频率（新增）

1. **功能描述**
   - 用户发布时通过滑块选择“情绪波形”（平稳/焦虑/狂喜/忧伤）
   - 首页卡片根据发帖人波形呈现不同频率呼吸灯律动

2. **交互逻辑**
   - 服务器按时间窗口（如 10 分钟）统计同频用户数量
   - 在“我的”页顶部提示：`此刻，远方有 N 个人正与你同频呼吸`
   - 采用弱提示设计（低饱和、可自动淡出），不打断浏览

3. **视觉规则**
   - 动画参数来自 `emotionProfiles.animationDuration`
   - 卡片边缘辉光来自 `emotionProfiles.glowColor`
   - 动画应支持低性能机型降级（关闭阴影，仅保留轻微缩放）

### 2.3 耳语距离模式（Whisper Proximity，新增）

1. **功能描述**
   - 用户在首页长按并缓慢拖动发布按钮，接近某条树洞卡片时触发“耳语”交互。
   - 这是“低压力、弱连接”的匿名互动，不需要评论或私信。

2. **交互逻辑**
   - 当按钮中心坐标与卡片区域重叠度超过阈值（如 35%）并停留 ≥ 1 秒：
     - 卡片轻微高亮/浮起
     - 触发一次 `wx.vibrateShort()`
     - 浮现该贴“隐藏心情词”
   - 耳语成功后，作者收到弱通知：`有人静静听了你的心事`。

3. **技术实现要点**
   - `movable-view` 的 `bindchange` 中持续采样按钮坐标
   - 用 `createSelectorQuery().boundingClientRect()` 获取卡片矩形
   - 做轻量碰撞检测（中心点命中 + 重叠面积阈值）
   - 做节流（如 80~120ms）避免高频计算卡顿

### 2.4 呼吸岛屿（The Breathing Island，新增）

1. **功能描述**
   - 在首页可拖动发布按钮上加入“长按即呼吸引导”能力。
   - 用户长按不松手时，按钮按 4-7-8 节律进行扩缩与明暗变化。

2. **疗愈逻辑（生理干预）**
   - 吸气 4 秒：按钮缓慢放大、颜色变亮
   - 屏息 7 秒：按钮保持稳定、光晕微弱脉冲
   - 呼气 8 秒：按钮回落缩小、颜色回归柔和
   - 连续 2~3 轮后可显著缓和紧张情绪

3. **技术实现建议**
   - `touchstart` 启动节律动画，`touchend/touchcancel` 立即停止
   - 使用 `wx.createAnimation` 或 CSS `@keyframes` 驱动 `transform: scale()`
   - 对低性能机型降级为仅颜色渐变，不做复杂阴影

### 2.5 雨落字里（Rain on Words，新增）

1. **功能描述**
   - 首页内容流可开启“雨天模式”，屏幕顶部下落半透明雨滴。
   - 雨滴撞击卡片区域时产生微小溅射，营造“被雨洗过”的安静氛围。

2. **交互细节**
   - 用户可在设置中开关该模式（默认关闭）
   - 手指滑动时可叠加“玻璃划痕”效果（轻量轨迹）
   - 被雨滴命中的文字色可短暂加深 5%~10%

3. **技术实现建议**
   - 在 `scroll-view` 之上叠加高层级 `canvas`
   - 粒子系统模拟雨滴，按卡片 `rect` 信息做碰撞判断
   - 与白噪音系统联动：雨天模式可启用轻雨声轨

## 3. 我的页面（Profile Tab）

该页面侧重个人数据展示和个性化设置。

### 3.1 页面布局

1. **自定义顶部 Tab**
   - 风格与首页保持一致
   - 用于快速跳转设置、消息通知

2. **用户信息区（User Header）**
   - 背景支持主题色渲染
   - 展示头像、昵称、树洞等级、获赞/关注数据

3. **内容区域（Content Area）**
   - 二级 Tab 切换（我的发布 / 我的收藏 / 草稿箱）
   - 列表样式与首页统一

### 3.2 解忧盲盒（Comfort Blind Box，增强）

1. **功能描述**
   - 用户可“投入一个烦恼”，并随机“抽取一个安慰”。
   - 安慰内容可来自系统文案库或匿名用户共享的温暖瞬间。

2. **交互流程**
   - 输入烦恼（可选）→ 投入盲盒 → 翻牌动画 → 获取安慰文案
   - 支持轻量反馈：`这句有被治愈` / `换一句`

3. **治愈设计原则**
   - 以“交换温柔”为核心，不展示强对抗型互动
   - 结果页采用低对比度、慢节奏动画，避免信息压迫

### 3.3 情绪调色盘 / 心情日历（Mood Palette，新增）

1. **功能描述**
   - 每天首次打开小程序时，用户可在 12 种主题色中选一个“今日心情色”。
   - 无需输入文字，降低表达门槛，鼓励持续记录。

2. **数据沉淀与展示**
   - 记录字段：`date`、`moodColor`、`themeId`、`note(optional)`
   - 在“我的”页展示心情日历 / 色块云（月视图）
   - 支持回看趋势：近 7 日、近 30 日颜色分布

3. **疗愈价值**
   - 通过颜色流动可视化“情绪会变化”这一事实
   - 帮助用户从长期视角看到自我修复的轨迹

## 4. 发布页面（Post Page）

由首页悬浮按钮点击跳转，是核心功能入口。

### 4.1 核心功能

1. **文本编辑**
   - 自动聚焦多行输入框（`textarea`）
   - 支持匿名切换

2. **媒体附件**
   - 支持上传 1~9 张图片
   - 支持语音/视频短片段

3. **主题关联**
   - 发布时可选择“心情标签”

4. **发布逻辑**
   - 发布前校验（含敏感词过滤）
   - 发布成功后返回首页并触发列表刷新

### 4.2 时空回声（Time Echo，新增）

1. **触发方式**
   - 用户拖动发布按钮进入底部隐藏区域并停留 2 秒
   - 触发震动反馈（`wx.vibrateShort`）并弹出“投掷回声”确认层

2. **功能逻辑**
   - 该树洞不立即公开，进入“沉睡期”
   - 由后端按 `releaseDelay`（24h / 72h / 7d）随机释放给陌生用户
   - 接收者可做一次“无名回应”（如 🌸 小花、🤗 拥抱）
   - 原作者在未来某时刻收到回声回应提醒

3. **数据字段建议**
   - `postMode`: `normal | time_echo`
   - `releaseDelay`: number（秒）
   - `releaseAt`: ISO 时间
   - `echoStatus`: `sleeping | released | replied`
   - `emotionWave`: number（用于联动情绪共鸣）

4. **前端呈现规范**
   - “时空回声”内容卡使用半透明/模糊滤纸效果
   - 添加角标：`来自过去的声音`
   - 交互以轻动效+弱文案传达，不使用强提醒红点

### 4.3 情绪色彩实验室（Color Chemistry，新增）

1. **功能描述**
   - 用户在发布页输入内容时，系统基于关键词做轻量情绪识别。
   - 发布按钮颜色实时“流光偏移”，体现“见字如面”的情绪反馈。

2. **交互逻辑**
   - 监听 `textarea` 的 `input` 事件
   - 根据命中词将按钮颜色向对应主题族偏移：
     - 忧伤/失落/深夜 → 蓝紫冷色系
     - 暖心/治愈/想念 → 橙粉暖色系
     - 平静/放空/呼吸 → 绿色低饱和系

3. **实现方式**
   - 前端维护轻量词库映射：`keyword -> emotionType -> targetPalette`
   - 对按钮使用动态内联样式：`background-color` + `box-shadow`
   - 颜色过渡用 `transition: background-color .35s ease, box-shadow .35s ease`
   - 兜底策略：未命中关键词时回退当前主题主色

### 4.4 “碎纸机”与“星空升华”（The Shredder & Ascension，新增）

1. **功能描述**
   - 用户写下高压力文本后可选择“粉碎”或“放飞”，而非立即发布。
   - 目标是提供“写下即释放”的可感知情绪出口。

2. **交互分支**
   - **粉碎**：文字被吸入底部碎纸区域，触发轻震动与碎纸声。
   - **放飞**：文字化作星点上浮至顶部星空并逐渐消散。

3. **技术实现建议**
   - 动画层：`canvas` 粒子动画（优先）或 CSS Keyframes（降级）
   - 定位：`wx.createSelectorQuery()` 获取 textarea 区域，覆盖临时动画层
   - 粒子策略：按文字区域采样为点阵，执行下坠/上浮路径
   - 结束态：清空输入框并反馈 `已释放`，不写入公开流

4. **触感与声音**
   - 粉碎触发一次 `wx.vibrateShort()`
   - 放飞触发轻反馈（可选）
   - 音效遵循用户“静音/降噪”设置，默认不过强

## 5. 技术细节梳理

| 功能点 | 技术选型/组件 | 备注 |
|---|---|---|
| 主题切换 | CSS Variables + `page-meta` | 便于统一管理 12 种风格 |
| 拖动按钮 | `movable-view` | `direction="all"` 并监听点击事件 |
| 组件化自定义 Tab | 自定义组件 | 首页和我的页面复用同一套 UI 逻辑 |
| 状态管理 | `mobx-miniprogram` 或 `globalData` | 主题切换后全站 UI 实时同步 |
| 情绪共鸣频率 | 发布页 slider + 首页卡片动画 | 波形参数驱动 `animation-duration` 与辉光 |
| 时空回声 | `movable-view` 坐标监听 + 延时发布字段 | 进入隐藏区停留触发，后端按 `releaseDelay` 调度 |
| 情绪色彩实验室 | 发布页 `input` 监听 + 词库映射 | 根据文本情绪动态改变发布按钮色彩 |
| 耳语距离模式 | `movable-view` + 卡片碰撞检测 | 重叠停留触发震动与“静静听了你”弱通知 |
| 碎纸机/星空升华 | Canvas 粒子动画 + Haptic | 将“负面倾诉”转化为可视化释放行为 |
| 潮汐白噪音环境 | 主题与音频映射 + 音量动态调节 | 以听觉包裹阅读/倾诉过程，降低焦虑感 |
| 解忧盲盒 | 随机文案服务 + 翻牌交互 | 形成“投入烦恼-获得安慰”的疗愈闭环 |
| 呼吸岛屿 | 长按按钮触发节律动画 | 通过 4-7-8 呼吸引导实现生理降压 |
| 雨落字里 | Canvas 雨滴粒子 + 文字碰撞反馈 | 以“雨洗”意象帮助用户从烦恼中抽离 |
| 情绪调色盘/心情日历 | 每日颜色打卡 + 月历聚合绘制 | 让用户看见情绪流动，增强恢复感 |

## 7. 接口与数据约定（建议）

### 7.1 发布接口（`POST /posts`）

请求体建议：

```json
{
  "content": "今天有点累，但还是想再坚持一下",
  "anonymous": true,
  "emotionWave": 72,
  "emotionType": "anxious",
  "postMode": "time_echo",
  "releaseDelay": 86400,
  "media": []
}
```

### 7.2 同频统计接口（`GET /resonance/current`）

响应体建议：

```json
{
  "emotionType": "anxious",
  "windowMinutes": 10,
  "resonanceCount": 38,
  "hintText": "此刻，远方有 38 个人正与你同频呼吸"
}
```

### 7.3 耳语投递接口（`POST /whispers`，建议）

请求体建议：

```json
{
  "postId": "p_123456",
  "action": "listen_quietly",
  "hoverMs": 1180,
  "overlapRatio": 0.41
}
```

响应体建议：

```json
{
  "success": true,
  "notifyText": "有人静静听了你的心事"
}
```

### 7.4 解忧盲盒接口（`POST /comfort/box`，建议）

## 8. 验收标准（新增功能）

1. 用户可在发布页稳定选择情绪波形，提交后可被正确保存。
2. 首页卡片可根据不同情绪显示差异化呼吸律动。
3. 同频提示仅在满足统计条件时出现，且不打断主流程。
4. 拖动发布按钮到隐藏区停留 2 秒可稳定触发“时空回声”。
5. “时空回声”帖子会按延迟策略释放并可收到匿名回应。
6. 发布页输入文本时按钮颜色可在 300~500ms 内平滑变化且可回退。
7. 首页耳语命中后触发一次触感反馈，并向作者投递弱通知。
8. 用户可完成“粉碎/放飞”释放流程，动画与触感反馈正常，且不产生公开帖子。
9. 主题切换后白噪音可正确联动，播放/暂停与音量微扰策略生效。
10. 解忧盲盒可稳定返回安慰内容，支持“换一句”与“被治愈”反馈。
11. 呼吸岛屿可在长按时稳定执行 4-7-8 节律，松手即停止且不影响拖动。
12. 雨落字里模式下帧率与滚动保持可接受（目标 ≥ 45fps），并支持随时关闭。
13. 心情日历可正确展示当月色块与趋势统计，不漏记、不串日。

## 9. 疗愈交互实现备注（开发者）

1. **碎纸机粒子化思路**
   - 通过 `wx.createSelectorQuery()` 获取 `textarea` 位置与尺寸
   - 在同位置挂载临时 canvas 层
   - 将文字区域采样成粒子点阵，执行坠落/吸入动画
   - 动画完成后卸载 canvas，恢复页面交互

2. **性能建议**
   - 粒子数量分级：低端机 80~120、中高端机 180~260
   - 碰撞检测与滚动联动务必节流/防抖
   - 所有高频动画均应提供“低性能模式”开关

3. **心理安全边界**
   - 提示文案避免评判式表达，统一使用陪伴型语气
   - 所有疗愈反馈均可关闭，尊重用户情绪自主权

4. **功能清单（疗愈增强版）**

| 功能名称 | 页面入口 | 核心治愈逻辑 | 技术关键点 |
|---|---|---|---|
| 呼吸岛屿 | 首页拖动按钮 | 生理干预：呼吸引导降低焦虑 | `touchstart/touchend` + 自定义 animation 节律 |
| 碎纸机/星空升华 | 发布页 | 心理卸载：亲手释放负面情绪 | Canvas 粒子动画 + Haptic + 音效 |
| 潮汐白噪音联动 | 全局主题 | 感官沉浸：营造安全宁静氛围 | `wx.createInnerAudioContext` + 主题映射 |
| 心情日历 | 我的页面 | 时间疗愈：可视化情绪流动与转变 | 每日打卡聚合 + 日历绘制（Canvas/组件） |

### 11.3 核心组件开发逻辑

#### A. 呼吸岛屿（可拖动 FAB）
- `movable-view` 实现自由拖动。
- 监听 `longpress` 启动呼吸动画（`scale 1.0 -> 1.2 -> 0.9`）。
- 动画阶段触发轻触感反馈：`wx.vibrateShort({ type: 'light' })`。

#### B. 碎纸机动画（Canvas 粒子化）
1. 获取 `textarea` 文本内容。
2. 点击粉碎时，在原区域覆盖临时 canvas。
3. 文本转图后通过 `getImageData` 提取像素。
4. 施加向下重力 + 随机水平速度，模拟碎纸飘落。

#### C. 雨落字里（沉浸式背景）
1. 顶层 canvas 绘制雨滴线段。
2. 维护 `rainDrops` 数组并逐帧更新。
3. 雨滴触达卡片顶部时触发水花并销毁当前粒子。

> - `pages/index/index` ≈ `pages/home/index`
> - `pages/mine/mine` ≈ `pages/profile/index`

#### 首页（`pages/home/index`）
- Top Bar：自定义组件，阴影随主题透明度变化。
- Content：`scroll-view` + `enable-back-to-top`。
- Floating Area：`movable-area` 全屏覆盖，承载 Breathing Island。

#### 我的（`pages/profile/index`）

- 解忧入口：盲盒弹性翻牌动效（Spring 风格）。

### 11.5 性能与优化建议

1. **音频常驻**
   - 使用全局 `InnerAudioContext`（或 `BackgroundAudioManager`）确保切页不中断。

2. **渲染压力控制**

3. **主题切换性能**

4. **动画降级策略**

## 12. 交互设计规范（Interaction Design Document）

本章节用于约束“进入-倾诉-离开”全链路体验，确保用户始终感到被包裹、被接住。

### 12.1 核心交互原则（Design Principles）

1. **无压力感**
   - 降低强制输入，优先采用滑块、长按、拖拽等低门槛动作。

2. **感官同步**
   - 视觉（主题色）/ 听觉（白噪音）/ 触觉（震动）必须统一节奏，不冲突。

3. **流动性**
   - 页面切换与状态过渡优先淡入淡出、柔和缩放，避免生硬推拉。

### 12.2 页面交互细则

#### A. 首页：沉浸式探索

- **自定义顶部 Tab**
  - 点击 Tab 时触发微波纹扩散，颜色取当前主题主色。
  - 列表上滑时 Tab 吸顶并提升背景透明度，让内容权重更高。

- **内容卡片**
  - 单击卡片展开全文；长按 1.5 秒进入耳语模式。
  - 耳语命中时触发轻震动，展示对方当前心情色块（弱提示层）。

- **底部 Toolbar**
  - 极简保留“白噪音播放/暂停”。
  - 图标可随音频节律轻微呼吸跳动。

#### B. 呼吸岛屿（悬浮发布按钮）

- **日常状态**
  - 按钮在中间区域可自由漂浮，拖动具有轻惯性。

- **长按激活**
  1. 长按 1 秒进入呼吸模式。
  2. 吸气阶段：按钮放大到 `1.2x`，触发长而弱的震动。
  3. 呼气阶段：按钮回落到 `0.9x`，震动停止。

- **松手释放**
  - 退出呼吸模式后回弹到最近停靠点。
  - 可出现温柔问句：`现在感觉好点了吗？要写点什么吗？`

#### C. 发布页：情绪洗涤

- **自动变色**
  - `textarea` 字数与情绪词共同驱动背景饱和度缓慢变化。

- **粉碎逻辑（重点）**
  - 在发布区提供“碎纸机”入口（拖拽触发或点击触发）。
  - 粉碎时：屏幕轻暗、文字粒子坠落、碎纸声 + 连续短震。
  - 完成后返回首页提示：`它们已经消失在风里了`。

#### D. 我的页面：情感沉淀

- **心情日历（调色盘）**
  - 建议蜂窝/网格布局。
  - 点击某日色块时，可短时切回当日主题白噪音，帮助情绪回溯。

- **解忧盲盒**
  - 支持“摇一摇”或“长按盲盒”触发。
  - 采用翻牌/抽签动效，安慰语以缓动方式浮现。

### 12.3 情感化反馈矩阵（V-A-H）

| 触发动作 | 视觉反馈 (V) | 听觉反馈 (A) | 触觉反馈 (H) |
|---|---|---|---|
| 切换主题 | 全场淡入新色调 | 白噪音 Crossfade 过渡 | 无 |
| 长按呼吸按钮 | 匀速扩缩 + 明暗变化 | 风声/潮汐声轻微增强 | 持续线性弱震动 |
| 粉碎文字 | 粒子坠落 + 页面微暗 | 碎纸机机械声 | 连续短促震动 |
| 成功发布 | 文字化星上浮 | 清脆铃声（Chime） | 成功反馈震动 |
| 开启雨落 | 全屏雨滴粒子覆盖 | 雨声增强 + 轻混响 | 随机极轻点震 |

### 12.4 异常与边界交互

1. **空状态**
   - 首页无内容时展示“海滩空贝壳”。点击后播放一段随机白噪音。

2. **加载中**
   - 不使用传统转圈，改为“呼吸岛屿”居中深呼吸动画。

3. **弱网提示**
   - 文案建议：`信号穿过森林需要一点时间，请稍等...`

### 12.5 给开发者的实现贴士（Implementation Tips）

1. **震动分级**
   - iOS 可使用 `wx.vibrateShort` 的 `type`（`light/medium/heavy`）。
   - 呼吸引导用 `light`，粉碎用 `medium`，成功发布用 `light`。

2. **音频无缝衔接**
   - 主题切换时先降低旧音轨音量再停播，再渐增新音轨音量（Crossfade）。

3. **Canvas 组件隔离**
   - “雨落”和“粉碎”应封装为独立组件，按页面按需挂载/卸载，防止内存泄漏。

4. **交互一致性**
   - 所有疗愈反馈均需提供开关，统一由 `settings/healing` 控制。

## 13. 女性向审美优化规范（Premium / Airy / Surprise）

本章节面向女性用户审美偏好，强调“高质感、呼吸感、细节惊喜”。

### 13.1 视觉风格策略

1. **微光卡片（Glow Card）**
   - 弱边框、低饱和、弥散阴影，避免生硬线条。

#### A. 首页（Tree Hole）

  - 弱化下划线，采用胶囊态或字重切换。
  - 背景可加极浅磨砂层，提升高级感。

  - 非对称交错布局（在可控范围内），缓解视觉疲劳。

#### B. 我的页面（Personal）

- 情绪调色盘：
  - “色块星球”风格，点击有轻微弹性回馈。

### 13.3 交互细节优化

1. **呼吸岛屿（FAB）女性化改造**
   - 形态不固定为硬圆，拖动时可做轻微拉伸（液滴/云团质感）。
   - 到屏幕边缘时增加“果冻回弹”反馈（视觉 + 轻触感）。

2. **粉碎动作柔化：花瓣雨 / 泡泡消散**
   - 将“粉碎”扩展为美感分支：
     - 樱花瓣上升消散（偏粉主题）
     - 泡泡漂浮淡出（偏蓝/绿主题）
   - 心理暗示从“摧毁”转为“温柔化解”。

## 14. 双风格交互与视觉方案（Dual-Aesthetic）

本章节用于在同一产品内兼容两类情绪审美偏好：
- 前 6 套主题：**柔美愈系（Soft & Airy）**
- 后 6 套主题：**理性深邃（Structured & Deep）**

### 14.1 视觉语言对照

#### A. 柔美愈系（感性）
- 核心词：呼吸、圆润、通透、低饱和。
- 形状：超圆角（`24rpx+`）或有机形（云朵/卵石）。
- 投影：轻弥散阴影，模拟薄雾光感。
- 材质：磨砂玻璃（Frosted Glass）+ 柔和渐变。
- FAB 手感：气泡感拖拽，伴随微弱水波纹。

#### B. 理性深邃（结构）
- 核心词：极简、边界、深色调、工业感。
- 形状：标准圆角（`8rpx~12rpx`），几何线条明确。
- 层级：弱化投影，使用细描边（Stroke）区分模块。
- 材质：纯色块/深色渐变（夜色、矿石、深海意象）。
- FAB 手感：金属球/陨石感拖拽，惯性反馈更干脆。

### 14.2 关键交互差异化映射

| 功能 | 柔美愈系（舒缓压力） | 理性深邃（释放压力） |
|---|---|---|
| 粉碎动作 | 花瓣消散：樱花/羽毛上浮消失，配微风声 | 碎裂击穿：文本瞬时裂解，配清脆破裂声 |
| 呼吸岛屿 | 起伏扩张：柔和明暗变化 | 脉冲收缩：更规律的机械脉冲感 |
| 雨落字里 | 细雨朦胧：雨点细密、文字轻化开 | 雷雨阵列：雨滴更快，偶发微闪光提示 |
| 解忧盲盒 | 抽礼盒：丝带解开、手写文案浮现 | 开机密：解锁/保险箱动效、打字机文案 |

### 14.4 布局适配建议

1. **首页内容密度**
   - 柔美愈系：更大卡片间距、行高约 `1.8`、更多留白。
   - 理性深邃：更紧凑的信息密度、细线分割、更强秩序感。

2. **我的页情绪可视化**
   - 柔美愈系：花海/色块星球。
   - 理性深邃：频谱图/热力方块。

### 14.5 自然切换机制（不增加用户压力）

1. **默认策略**
   - 首次进入使用中性主题，不强制选择性别/风格。

2. **行为触发（隐式）**
   - 用户偏向柔美主题时，系统逐步增大全局圆角、提升柔光动画占比。
   - 用户偏向深邃主题时，系统逐步收紧圆角、增强描边和结构化动效。

3. **手动控制（显式）**
   - 在“我的-设置”提供 `视觉偏好`：`柔和风 / 硬朗风 / 跟随主题`。

### 14.6 设计与伦理说明

- “双风格”用于匹配审美与情绪偏好，不用于限定用户身份。
- 所有风格能力均可自由切换，避免刻板化体验。
- 推荐在埋点中仅记录“风格偏好”，不记录敏感身份标签。

## 15. PRD（执行版汇总）

### 15.1 项目概述

- **项目名称**：某某树洞（待定）
- **产品定位**：兼具匿名社交与情绪疗愈的轻量空间
- **核心目标**：打造安全、美观、低心理门槛的倾诉环境

### 15.2 视觉与主题规范（Theme Architecture）

系统支持 12 种主题，分两组：
- `0-5`：柔美感性（偏女）
- `6-11`：理性深邃（偏男）

#### 主题字段规范（`theme.config.js`）
- 颜色：`primaryColor` / `bgColor` / `cardColor` / `textColor`
- 交互参数：
  - `borderRadius`（柔美建议 `40rpx+`，深邃建议 `12rpx~20rpx`）
  - `shadowStyle`（弥散阴影 or 线性描边）
  - `audioSrc`（主题白噪音）
  - `particleType`（`petal` / `shatter`）

### 15.3 页面详细需求

#### A. 首页（Home Tab）
- 顶部 Tab：推荐/附近/最新；支持左右切换，背景微渐变过渡
- 中部内容区：
  - 柔美：错落卡片
  - 深邃：对齐方块
- 底部 Toolbar：白噪音开关 + 主题快速预览
- FAB：全局 `movable-view`，长按触发呼吸岛屿（含 `wx.vibrateShort`）

#### B. 我的页面（Profile Tab）
- 用户区：异形头像 + 昵称 + 心情签名
- 情绪调色盘：展示近 30 天色块；点击回听当天白噪音
- 列表区：我的发布 / 我的收藏 / 解忧记录
- 解忧盲盒：翻牌/抽奖动效 + 治愈文案

#### C. 发布页面（Post Page）
- 沉浸输入：`textarea` 自动聚焦，背景随字数/关键词变化
- 发布动效：文字星点上浮
- 粉碎动效（重点）：
  - 柔美：花瓣消散
  - 深邃：玻璃碎裂
- 隐私：默认匿名，可选定位与心情标签

### 15.4 六大治愈功能（The Healing Six）

| 功能 | 触发场景 | 交互表现 | 心理价值 |
|---|---|---|---|
| 粉碎 | 发布页 | Canvas 粒子销毁 + 碎裂/风声 | 负面情绪卸载 |
| 解忧盲盒 | 我的页 | 震动 + 暖心文案浮现 | 非即时情感支持 |
| 情绪调色盘 | 我的页 | 30 天色彩可视化 | 看到情绪流动周期 |
| 呼吸岛屿 | FAB | 4-7-8 动画 + 节律触感 | 生理性降低焦虑 |
| 雨落字里 | 首页/发布页 | 雨滴碰撞文字 + 水花 | 白噪音式隔离感 |
| 潮汐白噪音 | 全局 | 主题绑定循环自然音 | 进入沉浸状态 |

### 15.5 UX 标准

1. **自然性**：统一使用缓动曲线（如 `cubic-bezier`），避免突兀跳变。
2. **触感反馈**：核心动作（粉碎/盲盒/呼吸）必须有触感层。
3. **简洁性**：
   - 柔美：留白 + 毛玻璃 + 弱边界
   - 深邃：高对比 + 清晰边界 + 几何秩序
4. **性能**：Canvas 页面隐藏后自动停渲染并释放资源。

### 15.6 非功能需求

1. **兼容性**：适配 iOS/Android，不同全面屏安全区避让。
2. **内容安全**：接入微信官方内容安全/敏感词能力。
3. **离线缓存**：主题与白噪音设置写入 Storage 并自动恢复。

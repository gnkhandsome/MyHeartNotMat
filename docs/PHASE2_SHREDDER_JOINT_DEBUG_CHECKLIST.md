# Phase 2.2 粉碎能力联调清单（Canvas 版）

## 一、主题风格联调

- [ ] female 主题下触发粉碎，粒子表现为 **petal**（花瓣漂浮）
- [ ] male 主题下触发粉碎，粒子表现为 **shatter**（碎裂散射）
- [ ] 主题切换后无需重启页面，下一次粉碎粒子风格即时生效
- [ ] 粒子颜色跟随当前主题 `primaryColor`

## 二、功能联调（Home / Post）

### Home
- [ ] 发布输入区粉碎：震动 + 音效 + Canvas 粒子 + 清空 + 返回首页
- [ ] 三个流卡片粉碎：我的小屋 / 广场 / 最新 均可删除
- [ ] 连续快速点击粉碎按钮，未出现重复叠加动画（并发防抖生效）

### Post
- [ ] 输入区粉碎：震动 + 音效 + Canvas 粒子 + 清空（不入流）
- [ ] 已发布内容卡片粉碎后删除成功
- [ ] 心情日记卡片粉碎后删除成功

## 三、性能保护联调

- [ ] 在低性能设备上，粒子数量自动降级（无需业务侧改动）
- [ ] 动画总时长受保护（超时兜底，不会卡死）
- [ ] 高频触发下页面无明显掉帧、无明显内存抖动
- [ ] 无法获取目标节点时，动画优雅失败且不阻断主流程

## 四、异常与边界联调

- [ ] 空输入时点击粉碎，仅提示，不触发动画
- [ ] 删除接口失败（可模拟）时，页面不崩溃、状态可恢复
- [ ] 切后台再回来后，粉碎功能仍可正常使用
- [ ] 音频权限受限时，粉碎动画仍可独立执行
